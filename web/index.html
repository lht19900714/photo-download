📖 如何获取 GitHub Token？

  1. 访问 GitHub Token 设置页面
  2. 填写 Note: PhotoPlus Downloader
  3. 设置过期时间（建议选择 No expiration）
  4. 勾选权限:
    • repo (完整仓库访问)
    • workflow (触发 Actions)
  5. 点击 "Generate token" 并复制 Token
